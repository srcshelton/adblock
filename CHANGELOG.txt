UNRELEASED:

1.8.0.1:

1.8.0.0:
- Added log messages to the AdBlock pop up menu to help determine usage of each menu item.  (#117)

1.7.1.0:
- Fixed an issue with the build process to include all necessary files.
- Updated translations.

1.7.0.0:
- Changed the filter list subscription persistence logic to not store EasyList Lite to persistent storage, and instead load it from a file included with the add on.  EasyList Lite will be updated 1 hour after the browser has been started.  (#110)
- Fixed the updating of the frame data information when tracking navigation on single page sites that use the History API.  This will allow the YouTube channel white-list feature to function correctly on Edge. (#112)
- Added 'bandaids' processing to prevent sites from opening a new tabs certain sites.  (#113)

1.6.1.0:
- Fixed a bug with the 'getting' of the STATS unique identifier. (#108)

1.6.0.0:
- Added a log message when the popup menu is opened.  (#107)
- Removed old Safari specific browser logic.  (#102)
- Implemented the improved ping / stats processing.  (#103)

1.5.0.0:
- Removed the 'bandaids' processing on YouTube because of video playback issues.  (#99)
- Added a check for invalid characters to the purge element processing to avoid issue with bad CSS query strings.  (#98)
- To prevent memory leaks, the blocked resource cache now has an max size of 1,000 entries.  (#77)

1.4.0.0:

- Added 'bandaids' processing to help block streaming ads on YouTube.  (#84)
- Fixed a JavaScript error on the 'Show All Requests' page.  (#89)
- Added support of the new 'websocket' filter list option. (#86)

1.3.0.0:

- Updated the Indonesian filter list URL.  (#67)
- Added support of the generic hide and generic block filter list options.  (#73)
- Changed the localized description string in the locale files.  (#80)

1.2.0.0:
